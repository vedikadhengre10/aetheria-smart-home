# 🏠 Aetheria Smart Home IoT Dashboard

A futuristic Smart Home Automation Dashboard that allows users to monitor, control, and automate connected home devices through an interactive web interface. Built with modern UI design principles, real-time simulations, energy analytics, security monitoring, and automation management.
